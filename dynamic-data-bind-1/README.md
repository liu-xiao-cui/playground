## 动态数据绑定（一）

实现一个 `Observer`，要求如下：

1. 传入参数只考虑对象，不考虑数组；
2. `new Observer` 返回一个对象，其 `data` 属性能访问到传递进去的对象；
3. 通过 `data` 访问和设置属性时，能在控制台打印出相应的信息。

题目来自于 IFE：http://ife.baidu.com/course/detail/id/15
